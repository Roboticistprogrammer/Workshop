# Workshops Projects

## Project Obejctives:

01 – FPS Performance (threading vs no threading)
Imagine you are watching a movie and also drawing on each picture to make it fancy. Doing one picture at a time is slow (single worker). This project shows how using two helpers at once (threads) lets one helper grab the next picture while the other helper draws, so the movie looks smoother and faster.

02 – ORB Feature Tracker
Think of picking a toy in a video and asking the computer, “Follow this toy wherever it goes.” The project lets you draw a box around the toy, then the computer looks for the same tiny patterns on the toy in every new picture and keeps drawing a box around it as it moves.

03 – Multiprocessing Vision Workers
Pretend you have a big pile of pictures and you want to blur them to make them look soft. If one kid does all the blurring, it’s slow. This project shows how to give different pictures to many kids (many processes) at the same time so they can all blur together and finish faster while still showing you the video smoothly.

04 – Asyncio and Concurrency
Imagine you have to wait for many timers to finish (like lots of little hourglasses) and also do some heavy math homework. This project shows when it’s smart to handle many timers at once (async, good for waiting tasks) and when you need more brains (extra processes) to do hard math faster.
